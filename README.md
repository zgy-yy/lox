# Lox

一个使用 Vite 构建的项目

## 安装

```bash
npm install
```

## 开发

启动开发服务器：

```bash
npm run dev
```

开发服务器将在 `http://localhost:3000` 启动。

## 构建

构建生产版本：

```bash
npm run build
```

构建产物将输出到 `dist/` 目录。

## 预览

预览生产构建：

```bash
npm run preview
```

## 技术栈

- Vite - 下一代前端构建工具
- TypeScript - 类型安全的 JavaScript 超集
- ES Modules - 现代 JavaScript 模块系统
